# instalação de ambiente virutal no python
Seguir os passos do link.1 para instalar o virtualenv e depois instalar as libs abaixo:

- pip install virtualenv
- cd ..
- virtualenv -p python3 pyPDFweb
- source bin/activate
- pip install selenium
- pip install webdriver_manager
